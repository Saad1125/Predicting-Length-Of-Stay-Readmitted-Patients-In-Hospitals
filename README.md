The length of stay (LOS) has a direct bearing on
patient outcomes, resource use, and hospital revenue, making it
a crucial element in healthcare administration. Accurate LOS
prediction can aid hospitals in discharge planning and resource
allocation. In recent years, LOS prediction models have been
created using machine learning approaches. Given that hospital
readmission is linked to higher healthcare costs and poorer
patient outcomes, it is a major concern over healthcare providers.
Hospitals can target interventions and reduce readmissions by
predicting how many patients are in danger of doing so. Consid
ering the data resources used, the machine learning strategies
used, and the performance measures employed to assess the
models, this work seeks to provide an overview of the most
recent state of the art in LOS prediction. We examined the LOS
